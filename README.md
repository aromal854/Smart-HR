# Smart-HR

**Description**:  
Smart-HR is a minor project developed as part of the MCA program. The project is implemented entirely in JavaScript.

---

## Features

- User-friendly HR management system.
- Streamlined workflows for HR tasks.
- Interactive and responsive design.

---

## Technologies Used

- **JavaScript** - 100%

---

## Installation Guide

1. Clone the repository:

   ```bash
   git clone https://github.com/aromal854/Smart-HR.git
   cd Smart-HR
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the application:

   ```bash
   npm start
   ```
